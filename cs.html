Camstagram
Fall Semester 2016 – present

My final project for CIS 196: Ruby on Rails Web Development at Penn. It mimics Instagram with a lot of the major features, including user profiles, posting images, liking images, commenting, following users and being followed by others. The majority of the work was utilizing erb to display the right informaiton in the right places (limiting the number of images displayed at any given time, ensuring a user only sees the posts of the people which he or she follows, etc.).

Major Features:
Hosted on Heroku. Images hosted via the Paperclip gem in tandem with aws s3. Passwords protected via bcrypt. Much of the formatting and styling was done through CSS modifications of Boostrap.

Things to Add:
Make the app more responsive to different screen sizes, particually mobile. Add more modern display for navbar & other links. Implement searching for users rather than just having a user show page. Integrating AJAX for real-time commenting and liking.

Minesweeper
Fall Semester 2016

My final project for CIS 120: Programming Languages and Techniques I at Penn. It is coded entirely in Java utilizing Java's Swing library for the GUI display.

Major Features:
Game board which automatically resizes the window based on different sizes. There are preset difficulties and the user can enter a custom difficulty. The game utilizes a random algorithm to generate the board based off of the user's initial click. Boxes touching zero mines utilize a recursive formula to reveal adjacent boxes. The game keeps track of time of the game and logs highscores based on difficulty and time for the game to be completed.
